📊 DATOS EN BASE DE DATOS

Nuevas columnas en tabla "egresados":
  fecha_proximo_recordatorio DATETIME  (cuándo mostrar próximo recordatorio)
  recordatorio_visto TINYINT(1)        (fue visto el recordatorio?)
  porcentaje_completitud INT           (% de campos completados)

Campos considerados para completitud:
  • Information: nombre, correo_personal, telefono, especialidad
  • Labor: empresa_actual, puesto_actual, modalidad, jornada, contrato
  • Skills: habilidades, experiencia

🔗 ENDPOINTS API

GET: Obtener estado actual
  /AppEgresados/public/api/marcar-recordatorio.php
  Retorna: { success, estado { debe_mostrar, porcentaje_completitud, ... } }

POST: Marcar recordatorio como visto
  /AppEgresados/public/api/marcar-recordatorio.php
  Body: { "accion": "marcar_visto" }
  Retorna: { success, proximo_recordatorio }
